# realpath

A realpath written in golang, mostly for macOS.

## Usage

```
$ realpath <your-file>

absolute-path-to-your-file

```

## Installation

```
$ go get github.com/lnmpy/realpath
```

or

```
$ git clone git@github.com:lnmpy/realpath.git
$ cd realpath && make install
```
